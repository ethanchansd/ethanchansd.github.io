PROJECTS

What I've done

Data Structures Companion

A native iOS app that helps students learn introductory Data Structures and Algorithms. It provides visualizations and explanations for lessons on their mobile device. This was developed with the support of University of Waterloo Professor: Igor Ivkovic and is developed specifically for a course used across multiple Waterloo Engineering Disciplines.

NoCaptcha

This is a hardware version of ReCaptcha more commonly known as 'Are you a robot'. It checks liveliness through a mathematical operation. An LED matrix and hardware encrypter was built from scratch. This was all controlled with an Arduino Uno with all native functionality re-implemented in an optimal fashion.

Elevator Control

An algorithm that determines the routing of an elevator system with an optimal destination time for passengers. Design in collaboration at Waterloo Engineering Competition.

Puzzle Design

Designed in SolidWorks, this 3D printed puzzle was envisioned with a custom track to affix pieces to a center core.
